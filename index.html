Program Nilai Siswa dengan Array

Web ini dirancang sebagai contoh aplikasi sekolah yang menampilkan proses pengolahan nilai siswa menggunakan struktur data Array. Anda dapat mengisi nilai secara manual, atau mengacak nilai secara otomatis untuk simulasi.

Penjelasan singkat:

  • Data disimpan dalam array dengan 30 elemen.
  • Perulangan for digunakan untuk hitung jumlah, rata-rata, min, dan max.
  • Nilai tertinggi & terendah dihitung secara langsung dari isi array.

Cara Menggunakan

  1. Tekan tombol Acak Nilai untuk mengisi 30 nilai siswa secara otomatis.
  2. Klik Hitung Statistik untuk melihat rata-rata dan nilai ekstrim.
  3. Buka menu Login untuk mencoba fitur akses pengguna.
